## 3. 后端变更

- 通知服务增加实时事件：
  - `notification.created`
  - `notification.read`
  - `notification.unread.changed`
- 事件 worker 在通知落库后推送给接收人。
- 新增 BUG 验证记录表 `issue_verification_logs`。
- 新增事项验证接口：`POST /api/issues/{issueId}/verifications`。
- 新增搜索索引重建接口：`POST /api/search/reindex`。
- 更新本地测试数据重置脚本，清理顺序包含 `issue_verification_logs`。

## 4. 前端变更

- 通知页面订阅 WebSocket 通知事件，收到创建、已读、未读数变化后刷新通知数据。
- 项目事项详情增加“验证记录”区域：
  - 仅 BUG 显示提交验证表单。
  - 支持验证通过、验证失败、验证阻塞。
  - 展示验证人、结果、时间和备注。
