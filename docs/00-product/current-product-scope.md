# 当前产品范围

Colla Platform 当前是一个面向研发团队的轻量协同平台。产品形态以 Web 为主，IM 是核心入口，同时已经具备项目、文档、表格、审批、通知、搜索、设备和管理等模块的 MVP 能力。

## 当前可用入口

当前 Web 路由来自 `web/src/app/router.tsx`：

| 入口 | 路径 | 当前能力 |
| --- | --- | --- |
| 登录 | `/login` | 用户登录，JWT token 写入本地状态 |
| 工作台 | `/` | 仪表盘、未读消息、未读通知、我的事项、审批待办、最近对象 |
| IM | `/im` | 单聊、群聊、会话列表、消息发送、编辑、撤回、置顶、表情、已读、成员管理、链接卡片 |
| 项目 | `/projects`, `/issues/:issueId` | 项目、事项、需求/任务/BUG、评论、附件、状态流转、统计、BUG 验证记录 |
| 文档 | `/docs`, `/docs/:docId` | 文档列表、编辑、块模型、评论、版本、权限、关系 |
| 表格 | `/bases`, `/bases/:baseId/...` | Base、表、字段、记录、筛选、排序、看板、日历视图 |
| 审批 | `/approvals`, `/approvals/:approvalId` | 审批表单、发起、审批、拒绝、转交、撤回、待办和统计 |
| 通知 | `/notifications` | 通知列表、筛选、未读数、标记已读、全部已读、目标跳转 |
| 设备 | `/devices` | 设备列表、撤销设备、登记测试 push token、push probe |
| 搜索 | `/search` | 全局搜索，覆盖事项、文档、表格记录、消息 |
| 管理 | `/admin/users`, `/admin/roles` | 用户管理、启停、重置密码；角色入口当前复用用户管理页面 |

## 已实现产品能力

### 身份与团队

- 默认 workspace 和内置管理员初始化。
- 登录、刷新、退出、当前用户查询。
- 管理员创建成员、启用/禁用用户、重置密码。
- 设备登录记录、设备撤销、push token 登记和测试。

### IM

- 支持 `group`、`direct`、`project` 类型会话。
- 支持单聊创建、群聊创建、成员添加/移除、退出群聊、关闭单聊。
- 支持置顶、静音、会话右键菜单。
- 消息支持 `clientMessageId` 幂等、`messageSeq` 补拉、编辑、撤回、置顶/取消置顶、表情 reaction。
- 粘贴内部链接可解析为对象卡片。
- WebSocket 推送消息、会话、未读和通知变化。

### 项目与 BUG 管理

- 项目成员与项目群联动。
- 事项类型包含 requirement、task、bug。
- 事项支持负责人、优先级、截止日期、状态流转、评论、附件、活动记录。
- BUG 支持验证记录；已解决 BUG 验证通过后可自动关闭，验证失败可退回处理中。
- 项目页面提供统计、看板和列表视图。

### 文档

- 文档支持创建、更新、目录移动。
- 支持结构化 blocks 读取和保存。
- 支持文档版本、版本 diff、版本恢复。
- 支持文档权限、关系、评论和评论提及通知。

### 多维表格 Base

- 支持 Base、Table、Field、Record、View。
- 当前字段类型以 MVP 为主，支持结构化记录值。
- 支持记录查询、筛选、排序、看板视图、日历视图和记录详情。
- Base/Table/Record 已接入平台对象和搜索。

### 审批

- 支持审批表单、流程节点、审批实例、审批任务、审批动作日志。
- 支持提交、审批、拒绝、转交、撤回。
- 审批事件可写入通知。

### 通知、搜索和平台对象

- 通知支持列表、未读数、已读、全部已读。
- 通知创建、已读和未读数变化可推送到 WebSocket。
- 全局搜索覆盖事项、文档、表格记录、消息，并通过平台对象摘要返回权限态。
- 平台对象支持 summary、navigation、recent、favorite、internal link resolve。

## 当前明确 Gap

| 方向 | 当前差距 |
| --- | --- |
| 多端 | 后端和前端 SDK 已按多端考虑，但桌面端、移动端仍不是正式产品交付形态 |
| 文档协同 | 已有块编辑、评论、版本；尚未实现多人实时协同编辑 |
| Base 高级能力 | 尚未实现公式、复杂关联、自动化、权限到字段/视图级别 |
| 审计前端 | 后端有审计查询 API，当前 Web 还没有独立审计页面 |
| 通知矩阵 | 已覆盖部分 IM、项目、文档、审批事件；仍需继续补全 Base、审批细分、权限变更等通知矩阵 |
| 运维交付 | 本地 Docker 依赖和质量门禁可用；备份、恢复、发布清单仍需 M25 收口 |
